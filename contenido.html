INTRODUCCIÓN

BIENVENIDOS A GAJERCSS

En la actualidad la creación de sitios o páginas webs se han vuelto una necesidad muy grande en todos los ámbitos de nuestra sociedad, es por ello que se crea: GAJERCSS un framework que ha sido realizado por un grupo de estudiantes de la ULS de la cátedra de internet II. Diseñado y desarrollado con un conjunto de herramientas de código abierto con HTML y CSS.

El framework cuenta con un sistema de grillas con el que el usuario podrá adaptar o modificar las columnas de su sitio web, además cuenta con diversas opciones de menú donde puedes utilizar el diseño que más convenga a tu necesidad o el diseño que te sea más amigable.

También el framework contiene botones con diseño moderno para acciones en formularios, cuadros de diálogo y más con soporte para múltiples tamaños en donde podrás modificar y personalizar a tu gusto, así también tablas y formularios con un código fácil de entender y modificar, contiene tarjetas que son el contenedor de un contenido flexible y extensible. Incluye opciones para encabezados y pies de página, una amplia variedad de contenido.

No dejando de mencionar que además cuenta con un carrusel o slider que es un componente de presentación de diapositivas para recorrer los elementos como: imágenes, diapositivas de texto, etc.

¡Haz tu Sitio o Página web de una manera muy fácil y sencilla con GAJERCSS el framework que es tu mejor opción!

SISTEMA DE GRILLAS

El sistema de grillas esta creado utlizando flexbox ya que esto nos facilito el trabajo a la hora de escribir el código.

COMO FUNCIONA?

El framework GAJERCSS utiliza una serie de contenedores, como lo son las filas y columnas para acomodar nuestros texto, imàgenes y demàs contenido de una manera muy fácil. A continuacion se presenta un pequeño ejemplo de como se estructura.

1 FILA 3 COLUMNAS

1 FILA 3 COLUMNAS

1 FILA 3 COLUMNAS

El código correspondiente al ejemplo anterior se muestra a continuación:


                                 

    <div class="fila">
            <div class="co-pe-4">
                    1 fila de 3 columnas
            </div>
            <div class="co-pe-4">
                    1 fila de 3 columnas
            </div>
            <div class="co-pe-4">
                    1 fila de 3 columnas
            </div>
    </div>


Explicación:

La clase .contenedor es la que contendra todo nuestros contenedores y nuestro contenido dicho de otra forma ésta sera nuestra clase madre. Sin embargo si no se desea utilizar esta clase no afectará a la estructura de nuestro documento.

Dicha clase tiene un max-width de 1100px y un width del 100% esto no permite que se adapte a los 1100px, también tiene un height auto dando como resultado que se adapte al contenido que tiene dentro, margin auto su función es dejar un espacio a la derecha e izquierda del documento.

La clase .fila es la encargada de dar el tamaño a lo largo y ancho de nuestro documento esta clase si sera necesaria si no se desea tener algún error. Ésta clase abarca el 100% de nuestro contenedor y no se vera afectada si no utilizamos la clase madre.

La clase.co-es indispensable y cambia según el número que se le desea otorgar los números son del 1 al 12 y existen medidas para los dispositivos como lo son los celulares tablet o computadoras esas medidas las veremos más adelante.

Menú #1

LOGO
Inicio Acerca de Contáctanos Promociones


LOGO
Inicio Acerca de Contáctanos Promociones


Código

 

    
    <header class="barra1 barra-fijo">
        <div class="menu1">
            <div class="logo">LOGO </div>
                <i class="fa fa-bars fa-2x icono" aria-hidden="true"></i>
                <nav>
                    <a href="#">Inicio </a>
                    <a href="#">Acerca de </a>
                    <a href="#">Contáctanos </a>
                    <a href="#">Promociones </a>
                </nav>
        </div>
    </header>
Explicación:

La clase .barra tiene un width: del 100% permitiendo abarcar todo el contenido disponible del navegador, dicha clase tiene un height: del 53px lo cual genera una altura perfecta para agregar nuestro menú de navegación, también posee un background-color dando un bonito color verde limón.

Clase .barra-fijo ésta clase permite dejar el menú de navegación fijo en la parte superior, a medida que baje en el navegador el menu no se escondera siempre permanecera fijo. Los atributos que posee ésta clase son position: fixed , right: 0 , left: 0 , z-index: 1030 , top: 0.

Clase .logo nos permite agregar letras en la esquina superior izquierda de nuestro menú de navegación. Atributos de la clase: position: absolute , padding: 10px , color: white , font-size: 27px.

.icono Ésta clase tiene como objetivo darle estilos al icono de menú que se muestra al visualizar la página desde un dispositivo como tablet o celular, dicho icono se puede encontrar en la página de Font Awesome, este nos será de mucha ayuda cuando se muestre en un dispositivo pequeño porque el menú estara oculto hasta el momento de que presionemos sobre el icono.

Para que todo el menú funcione correctamente se necesitara descargar jquery-2.1.1 y se creara un archivo aparte con extención .js que contendra el siguiente código.

código

 
     $(".icono").click(function(){
        $("nav").animate({
        width: "toggle"});
     });
